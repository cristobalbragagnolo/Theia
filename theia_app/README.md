# Theia

Asistente Flutter para captura y análisis de morfometría geométrica con 32 landmarks.

## Documentación

- Guía completa (ES/EN): `docs/guia_theia.md`

## Desarrollo rápido

- Flutter >= 3.24
- Activos clave: `assets/detector_nano_fp32.tflite`, `assets/pose_medium_fp32.tflite`
- Ejecutar: `flutter pub get` y `flutter run`
